This an app that has got a database. After registration you can see the entire database.
By the way it has got many bounds for length of a username, password and email.

HOW TO USE IT?
To start with, you need to register in database (You simply need to
 insert you email, username, and password.).
Then you can log in by inserting your username and password to the login window.
If you logged in successfully, a window with the entire database will appear.
